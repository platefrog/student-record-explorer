# 변경 이력

## v1.0.7

- 학생 분석 화면의 상태 카드 구성을 `전체/창체/교과/행발` 4칸으로 재정리했습니다.
- 학생 선택 패널에 `학년 필터`, `반 필터`를 추가해 원하는 반을 빠르게 조회할 수 있도록 개선했습니다.
- UI 파인튜닝 과정에서 발생한 `app.py` 문자열/인코딩 깨짐 구간을 복구해 실행 안정성을 높였습니다.

## v1.0.6

- 하단 명함 카드 영역의 좌우 정렬과 타이포 계층을 개선했습니다.
- 저작권/안내 문구의 가독성을 높이고 모바일 폭에서도 레이아웃이 안정적으로 보이도록 조정했습니다.

## v1.0.5

- Added dependency audit for bundled analyzers and Kiwi model runtime checks.
- Added explicit analyzer setup guidance for optional MeCab installation path.

## v1.0.3

- 하단 정보를 Streamlit 기본 구성요소로 변경하여 HTML 코드가 그대로 노출될 가능성 제거
- `VERSION`을 빌드 버전의 단일 기준으로 사용하고 소스·산출물 버전 일치 검증 추가
- 이전 빌드 산출물을 안전하게 정리한 뒤 새로 만드는 클린 빌드 절차 추가
- ZIP·설치본과 SHA-256 빌드 정보의 일치 여부를 자동 검증
- UTF-8 손상, 대체 문자와 알려진 한글 깨짐 문자열을 배포 전 자동 검사

## v1.0.2

- 업데이트 설치 시 실행 중인 구버전을 자동으로 종료하도록 개선
- 중단된 설치나 수동 삭제 뒤 남은 프로그램 파일을 새 설치가 자동 정리하도록 개선
- 제거 시 실행 중인 프로그램을 먼저 종료하여 삭제 충돌 방지
- 사용자 분석 데이터는 업데이트와 제거 과정에서 보존

## v1.0.1

- Windows 배포판에서 `pyparsing`이 요구하는 표준 라이브러리 `unittest`가 누락되어 앱 화면이 열리지 않던 문제 수정
- 기존 v1.0.0 설치 위에 업데이트할 수 있는 패치 설치본 제공

## v1.0.0

학생부 탐색기의 첫 정식 공개 배포판입니다.

- 학생부 창체·교과세특·행발 전처리와 학생별 통합
- 빈도·TF-IDF 특징어·워드클라우드 분석
- 추출 키워드의 학생부 근거 문장 확인
- 커리어넷 학과 말뭉치 연계도와 공통·보완 키워드 분석
- 학생별 최상위 유사 학과 자동 선택
- 학생부 기반 불용어 검토 후보와 표현 통일 사전
- Windows 설치본과 무설치본 제공
- 사용자 데이터와 설치 파일의 저장 위치 분리
