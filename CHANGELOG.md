# 변경 이력

## v1.0.2

- 업데이트 설치 시 실행 중인 구버전을 자동으로 종료하도록 개선
- 중단된 설치나 수동 삭제 뒤 남은 프로그램 파일을 새 설치가 자동 정리하도록 개선
- 제거 시 실행 중인 프로그램을 먼저 종료하여 삭제 충돌 방지
- 사용자 분석 데이터는 업데이트와 제거 과정에서 보존

## v1.0.1

- Windows 배포판에서 `pyparsing`이 요구하는 표준 라이브러리 `unittest`가 누락되어 앱 화면이 열리지 않던 문제 수정
- 기존 v1.0.0 설치 위에 업데이트할 수 있는 패치 설치본 제공

## v1.0.0

학생부 탐색기의 첫 정식 공개 배포판입니다.

- 학생부 창체·교과세특·행발 전처리와 학생별 통합
- 빈도·TF-IDF 특징어·워드클라우드 분석
- 추출 키워드의 학생부 근거 문장 확인
- 커리어넷 학과 말뭉치 연계도와 공통·보완 키워드 분석
- 학생별 최상위 유사 학과 자동 선택
- 학생부 기반 불용어 검토 후보와 표현 통일 사전
- Windows 설치본과 무설치본 제공
- 사용자 데이터와 설치 파일의 저장 위치 분리
